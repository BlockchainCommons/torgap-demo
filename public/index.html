Demo: verify a text file signed by the onion service

Abstract

Besides the signing demonstration of the onion service, this service also contains a DID document which can be found here.
You can automatically install this service with a StackScript which can be found at https://github.com/BlockchainCommons/torgap-demo

Download a file and the signature

  • First, download the text file
  • Next, download the signature of the file signed by the onion service
  • Finally, you can verify the signature with the onion address

Verify the signature

1) First, clone the minisign tool: git clone https://github.com/BlockchainCommons/torgap-sig-cli-rust.git

2) Make sure you have Rust and Cargo installed: https://doc.rust-lang.org/cargo/getting-started/installation.html

3) Copy the downloaded files from the previous section into torgap-sig-cli-rust directory, and verify them:

cd torgap-sig-cli-rust
cargo run verify text.txt --onion-address fscst5exmlmr262byztwz4kzhggjlzumvc2ndvgytzoucr2tkgxf7mid.onion

Timestamp the signature

Additionally, you can choose to timestamp the signature and verify its timestamp with OpenTimestamps. A timestamp for our signature is already provided here.
You can take both files to our torgapped OpenTimestamp service and try to verify the timestamp at http://3xcaaswwserqnox56z7d7wifwyxm2jsdv4fniffym5jwncqx3qr7uzid.onion/
